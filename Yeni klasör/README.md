# Viptaksi (VIP Star)

Monorepo: admin panel, public website, backend API, and server utilities.

- **Quick start:** [README_FIRST.md](./README_FIRST.md)
- **Full setup:** [FULL_README_FIRST.md](./FULL_README_FIRST.md)

## Restore from backup

1. Copy each `**/.env.example` to `**/.env` (`backend`, `apps/admin`, `apps/website`) and fill in real values.
2. From the repo root: `npm install` in each app/package as needed (or follow README_FIRST).
3. Run services per root `package.json` scripts (e.g. `npm run dev`).
